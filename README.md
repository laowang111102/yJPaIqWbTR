# Python网上订餐管理系统 python783

## 项目概述

本项目是基于Python语言和Oracle数据库的网上订餐管理系统，提供了一套全面的源码、数据库文件、PPT以及设计文档。

## 技术栈

- **后端语言**：Python
- **数据库**：Oracle

## 功能模块

系统主要包括以下功能模块：

1. **登录注册**
2. **首页**
3. **菜品信息管理**
   - 添加/编辑/删除菜品
   - 菜品分类管理
4. **订单信息管理**
   - 订单查询
   - 订单状态管理
5. **用户信息管理**
   - 用户资料维护
   - 用户权限管理
6. **收发系统消息**
7. **积分管理**
   - 积分获取规则
   - 积分兑换
8. **管理员信息管理**
   - 管理员登录
   - 管理员权限配置

## 系统角色

- **普通用户**：可进行登录、注册、浏览菜品、下订单、管理个人信息等操作。
- **管理员**：拥有系统管理权限，可以管理菜品信息、订单信息、用户信息等。
